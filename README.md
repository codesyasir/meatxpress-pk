# MeatXpress.pk

A Next.js 14 ecommerce website for fresh fish, desi chicken, and eggs — delivering in Lahore.

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **State**: Zustand (cart)
- **CMS** (optional): WordPress + WooCommerce headless on `cms.meatxpress.pk`
- **Deploy**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your values.

## Pages
- `/` — Home page with hero, categories, best sellers, reviews
- `/shop` — Shop page with filters, search, sidebar
- `/product/[slug]` — Product detail with reviews
- `/cart` — Cart page
- `/checkout` — Checkout with COD + JazzCash
- `/contact` — Contact form
